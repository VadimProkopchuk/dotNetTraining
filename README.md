# dotNetTraining

.NET Training Samples 

## Overview

The solution includes seven lessons. Each lesson contain class libraries and presentation layer (e.g. console app).
The solution will be supplemented and updated.

## Technologies 

* .NET Core 3.x
* Entity Framework Core 3.x
* Core DI
* ASP.NET Core 3.x
* AJAX
